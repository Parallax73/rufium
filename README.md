keyboard-based pdf reader

This project uses https://github.com/ajrcarey/pdfium-render, you must provide the compiled binary library for your operating system.

1. Download the PDFium Library
.1 Go to https://github.com/bblanchon/pdfium-binaries/releases
.2 Download the package for your OS(e.g. pdfium-linux-x64.tgz for Linux).
.3 Extract the archive
.4 Copy the library into the root of this project (root folder)
